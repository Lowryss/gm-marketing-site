Notas de implementação:
- Usar `divide-x-0 sm:divide-x divide-white/20` no grid container — separadores via Tailwind `divide-x` (mais simples que absolute positioning, sem overflow issues)
- `whitespace-nowrap` nos parágrafos dos números — evita quebra de linha em "R$ 210 Mi+" em mobile (Pitfall 3)
- Labels com `text-white` puro (não `text-white/80`) — garante contraste WCAG AA para texto pequeno sobre #ff2a2a (Pitfall 4)
- `R$&nbsp;` — usar non-breaking space entre "R$" e o número para evitar quebra de linha entre prefixo e contador
